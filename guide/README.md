> 版本 v1.3.3 · Godot 4.7

## 🎮 基本操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动视角 | WASD | 前后左右平移 |
| 上升 | 空格 | 视角上移 |
| 下降 | Ctrl | 视角下移 |
| 旋转视角 | ← → | 平滑旋转 90° |
| 俯仰视角 | ↑ ↓ | 上下倾斜 |
| 缩放 | 鼠标滚轮 | 拉近拉远 |

## 🎨 像素画绘制

| 操作 | 按键 |
|------|------|
| 打开画板 | 背包中找到方块 → 右键 → Paint |
| 绘画 | 左键拖拽 |
| 取色 | 右键点击像素 |
| 笔刷大小 | 点 1px / 2px / 4px 按钮 |
| 填色 | Fill 按钮 |
| 复制/粘贴 | Copy / Paste 按钮（按当前高亮面操作） |
| 清空 | Clear 按钮 |
| 重置 | Reset 按钮 → 恢复默认灰色 |
| 保存到文件 | Save 按钮 |
| 从文件加载 | Load 按钮 |
| 应用到世界 | Apply 按钮 → 场上所有同款方块即时更新 |

画板是一个立方体展开图，6 个面同时可见：
- **金色边框** = 当前选中的面
- 鼠标悬停切换面
- 调色盘在底部

## ⚡ 电路系统

### 基础元件

| 方块 | 颜色 | 功能 |
|------|------|------|
| 电源 (Power) | 红橙 | 持续供电 |
| 导线 (Wire) | 铜色 | 传输电力 |
| 开关 (Switch) | 灰色 | 右键开关，OFF 断路 |
| 灯泡 (Lamp) | 米白 | 有电时发光 |
| 反转器 (NOT Gate) | 粉色+蓝箭头 | 输入有电→关，无电→当电源 |

### 反转器方向

- **粉色面** = 输入端（箭头后方）
- **蓝色箭头** = 输出端
- **右键** → 旋转方向
- 输入端导线有电 → 反转器关闭
- 输入端无电 → 反转器向蓝色方向输出电力

## 📦 蓝图系统（压缩建筑）

### 使用方法

| 步骤 | 操作 |
|------|------|
| 1. 进入蓝图模式 | 按 **B** 键 |
| 2. 选线框大小 | 按 **4**（4×4×4）或 **8**（8×8×8） |
| 3. 放置线框 | 鼠标左键点地面 |
| 4. 在线框内搭方块 | 正常放置/拆除方块 |
| 5. 压缩 | 鼠标右键点线框 → 微缩物品入背包 |
| 6. 退出蓝图模式 | 再按 **B** 键 |

### 使用蓝图物品

| 操作 | 效果 |
|------|------|
| 左键放置 | 展开还原（原建筑原地重建） |
| Shift + 左键放置 | 放置微缩 3D 模型（1×1 方块） |

## 💾 存档系统

| 操作 | 按键 |
|------|------|
| 保存 | 顶栏 Save 按钮 |
| 加载 | 顶栏 Load 按钮 |

- 弹窗列表显示所有存档
- 可命名、覆盖、删除
- 每个存档独立文件夹（`user://saves/slot_XXXX/`）
- 旧版 `save.json` 首次启动时自动迁移

## 🔧 功能方块一览

| 方块 | 颜色 | 功能 |
|------|------|------|
| 移动方块 | 蓝色 | 激活后向箭头方向移动 1 格 |
| 拐弯方块 | 绿色 | 移动方块经过时改变方向（消耗） |
| 生成器 | 紫色 | 箭头后方有方块时，向前方复制 |
| 推动方块 | 橙色 | 推动前方方块 |
| 消耗方块 | 红色 | 接触时销毁推动者 |
| 粘液方块 | 亮绿 | 粘住相邻方块一起移动 |
