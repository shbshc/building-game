## 3. 电力系统

- **供电**：Power 方块 BFS 蔓延电力
- **导电**：Wire 和 ON 状态 Switch 传递电力
- **显示**：导线带电→亮黄发光，灯泡带电→亮白发光，开关 ON→红色
- **交互**：右键 Switch 切换，实时更新网络
- **无距离限制**：电力沿导线无限传递
- **导线连接**：放置相邻导线自动生成连接梁

---

## 4. 贴图系统

- 入口：背包(E)→右键方块→绘图面板
- 6 面独立编辑：选面→单面 320×320 画布绘制
- 调色盘 24 色 + 取色器
- 1/2/4px 笔刷、Copy/Paste 跨面
- Apply 自动保存 PNG，下次打开自动加载
- 6-Surface 立方体渲染，TEXTURE_FILTER_NEAREST 像素风格

---

## 5. 技术要点

| 系统 | 实现 |
|------|------|
| 电力 BFS | power_system.gd，每 tick 更新，复用推动链 BFS 思想 |
| 导线 mesh | 中心 0.2 小方块 + 6 方向 0.05 连接梁 |
| 6 面渲染 | SurfaceTool 构建 ArrayMesh，每 Surface 独立材质 |
| 右键交互 | SWITCH 切换、功能方块旋转、普通方块贴图 |
| 存档 | JSON v5 + PNG 贴图文件 |
