# 功能方块系统 Phase 3 — 实施计划

> 日期：2026-07-18  
> 引擎：Godot 4.7 (GL Compatibility)  
> 语言：GDScript  
> 版本目标：v1.3.0

---

**Goal:** 在 Phase 2 基础上完善贴图系统、增加新功能方块、优化游戏体验。

**Architecture:** 延续 Phase 2 架构。新功能注册到 FuncType 枚举和 item_types.gd。行为逻辑集成到 main.gd tick 循环或 block_manager.gd。

---

## 候选功能

### 1. 贴图系统完善
- **6 面独立渲染**：当前已支持，需验证每个面方向正确
- **贴图预设库**：内置若干图案模板、一键应用
- **贴图导出/导入**：分享贴图文件
- **贴图浏览器**：在面板中预览已保存的贴图

### 2. 新功能方块
- **速度控制方块**：放在移动方块旁改变移动速度（加速/减速/暂停）
- **传感器方块**：检测前方是否有方块、输出信号
- **传送方块**：移动方块碰到后传送到配对位置

### 3. 体验优化
- **Undo/Redo**：撤销上一步放置/删除
- **区域选择**：框选复制粘贴
- **移动速度可调**：滑块控制 0.5s ~ 5s
- **音效**：放置、移动、推动音效
- **设置菜单**：视距、灵敏度、按键绑定

### 4. 多人联机
- 局域网联机
- 方块同步
- 聊天

---

## 优先级

| 优先级 | 功能 | 理由 |
|:--:|------|------|
| 🔴 高 | 贴图预设库 | 降低绘制门槛 |
| 🔴 高 | Undo/Redo | 高频需求 |
| 🟡 中 | 速度控制 | 扩展轨道玩法 |
| 🟡 中 | 区域选择 | 提升建造效率 |
| 🟢 低 | 传感器/传送 | 有趣但非核心 |
| 🟢 低 | 多人联机 | 工作量大 |

---

## 实施方式

建议继续使用 **subagent-driven development**：每 Task 独立子代理 + 规范审查 + 代码质量审查。
