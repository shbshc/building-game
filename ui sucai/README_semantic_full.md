# UI素材完整JSON语义版说明

每个tile均为16×16像素，编号从左到右、从上到下。JSON中包含坐标、分类、语义名称、具体含义、拼接组、九宫格角色、是否可重复等字段。

## 字段说明
- id/file/index/grid_col/grid_row/x/y：定位信息
- category：素材类型
- semantic_name：详细中文名称
- value：字母/数字值，非字体素材为空
- assembly_group：可组合素材的组名
- nine_slice_role：九宫格或条形拼接角色
- repeatable：是否可重复平铺
- description_zh：给无法识图AI看的具体说明
